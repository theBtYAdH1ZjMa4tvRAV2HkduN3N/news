      • sudo mkfs.exfat -n LABEL /dev/sdXn, change format
      • sudo fsck.exfat /dev/sdXn, format disk
    • by "disk"
      • find the partition
      • format it, type-="CUSTOM"
        • fill filesystem="exfat"
      • done
